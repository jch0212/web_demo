# 小说阅读器项目结构

## 📁 项目目录结构

```
web_demo/
├── 📄 package.json              # 项目配置文件
├── 📄 server.js                 # 后端服务器
├── 📄 Dockerfile               # Docker镜像配置
├── 📄 docker-compose.yml       # Docker Compose配置
├── 📄 .dockerignore            # Docker忽略文件
├── 📄 deploy.sh                # Linux部署脚本
├── 📄 deploy.bat               # Windows部署脚本
├── 📄 start.bat                # Windows启动脚本
├── 📄 README.md                # 项目说明文档
├── 📄 项目结构说明.md           # 本文件
└── 📁 public/                  # 静态文件目录
    ├── 📄 index.html           # 主页面
    ├── 📄 styles.css           # 样式文件
    ├── 📄 script.js            # 前端脚本
    └── 📁 images/              # 图片资源目录
        └── 📄 cover.jpg        # 小说封面（占位符）
```

## 🚀 功能特性

### 前端功能
- ✅ 响应式设计，支持桌面和移动端
- ✅ 小说封面展示页面
- ✅ 章节列表导航
- ✅ 翻页功能（上一章/下一章）
- ✅ 键盘导航支持（左右箭头键、ESC键）
- ✅ 现代化UI设计

### 后端功能
- ✅ Express.js服务器
- ✅ RESTful API接口
- ✅ 小说数据管理
- ✅ 章节内容API

### 部署功能
- ✅ Docker容器化部署
- ✅ Docker Compose编排
- ✅ 云服务器部署支持
- ✅ 自动化部署脚本

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Node.js, Express.js
- **部署**: Docker, Docker Compose
- **样式**: 响应式设计，现代化UI

## 📱 页面结构

1. **封面页面** - 展示小说信息、封面、简介
2. **章节列表页面** - 显示所有章节，支持点击跳转
3. **阅读页面** - 章节内容展示，翻页功能

## 🎨 UI设计特点

- 渐变背景设计
- 卡片式布局
- 悬停效果
- 响应式适配
- 简洁的白色线条图标（符合用户偏好）

## 🔧 自定义配置

### 修改小说内容
编辑 `server.js` 文件中的 `novelData` 对象：
- 修改小说标题、作者、简介
- 添加或修改章节内容
- 更换封面图片

### 样式自定义
编辑 `public/styles.css` 文件：
- 修改颜色主题
- 调整布局样式
- 自定义字体

## 🚀 部署方式

### 本地开发
```bash
npm install
npm start
```

### Docker部署
```bash
docker-compose up -d
```

### 云服务器部署
1. 上传项目文件到服务器
2. 运行 `./deploy.sh` (Linux) 或 `deploy.bat` (Windows)
3. 配置反向代理（可选）
